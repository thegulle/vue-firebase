# Vue Firebase Todo App

## Firebase Config

* Enter your firebase information in .env and .env.development file.
* Use mail authentication and create "todos" collection on your firebase project.

## Installation

```
npm install
```

For development mode

```
npm run dev
```

For Production mode

```
npm run build
```



## Techs

* Vue.js
* Firebase
* Tailwind CSS
